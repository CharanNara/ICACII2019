User Specific Semantic Search for Encrypted Data

Vadipina Amarnadh and Parisa Premchand Goud

ABSTRACT

Number of users in cloud are increasing day by day As privacy is one of the major issue for the cloud computing the data is encrypted before the outsourced datasets. The common utilization of data becomes difficult for encryption purpose. Further it becomes complicated to search the given keywords in encrypted datasets. Several schemes are proposed for encrypted data search based on keywords. However, keyword-based search schemes does not give the correct meaning that is exact semantic information of users retrieval, and cannot completely meet the users requirement with the search of keywords. Therefore, the design of a content-based search scheme and making the semantic search which is more effective and context-aware is a challenge. In this paper, we propose encrypted semantic search, a semantic search scheme based on the concept of hierarchy and the semantic relationship in the encrypted datasets. Encrypted semantic search uses two cloud servers. One is used to store the datasets and return the results to data users. The other one is used to compute the similarity scores between the documents and the query and send the matches to the first server.
