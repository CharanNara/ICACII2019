Student Classifier using K-Means Clustering

Shruti Petwal, Sunny John K, Vikas Gorla and Sandeep Singh Rawat

ABSTRACT

Nowadays the amount of data stored in educational databases is increasing rapidly. These databases contain hidden information which can be useful. The performance in competitive examinations is a turning point in the academics and career for all students. The performance in these exams is measured using many criteria; therefore it is essential to analyze this data to assess students' performance so as to identify their strengths and weaknesses. We can then take measures to improve the performance of students. The current approach to analyzing performance is based on the overall marks. This approach often fails to understand the student's marks and seldom helps in betterment of marks. In this project instead of just grouping students based on their total score, it groups students based on their need to improve in a certain section if the test. In the proposed solution, Data Mining is used to analyze student data and cluster the students into different categories. The data set has been prepared by collecting the Mock Test marks of final year students. The test considered has Logical Reasoning and Aptitude, Verbal and Coding sections. The section-wise percentage is calculated for each student using the marks obtained by them. Clustering is then performed on this data to divide each student into a particular cluster. Each cluster represents a topic or combination of topics towards which the student has to put more efforts to score better marks. Students belonging to similar clusters are then grouped together using Clustering Technique. This approach aims to make performance analysis and decision making easy and accurate for instructors and course administrators. It also helps students perform better by focusing on areas that they are weak at.
